Le concept

Avant d’investir quelques milliers d’euro pour développer une application complète, il est préférable de tester votre idée avec une page web statique expliquant le project ou un MVP (projet minimum viable).

Unloq met en relation des alumni de l’école de programmation ‘Le Wagon’ (les Unloqers) avec des porteurs de projets ou PME qui aimeraient lancer un projet, mais n’ont pas les ressources internes pour le faire eux-mêmes.

Unloq garanti la qualité des projets finis grâce à des moments de coaching privés pour chaque projet.

Tarifs

Le prix varie selon le site web que vous voulez.

Un site web statique pour tester votre idée reviendrait entre 100€ et 300€ selon les détails spécifiques de votre projet.

Le prix d’une application web dynamique se situe entre 500€ et 1.500€ selon la difficulté du projet.

Une fois que nous avons reçu votre projet, nous vous envoyons une offre de prix (sans engagement) sous 48h.

Le paiement se fait à la signature du contrat. La somme sera versée au Networqer lorsque vous recevez le projet fini.
